## 🧭 UX Research & Problem Statement
*   **The Problem:** Users often find food delivery apps overwhelming due to cluttered menus and confusing cart customization (adding extras/sauces).
*   **The Solution:** Designed a clean, minimalist 3-step checkout system with smart recommendation chips based on user preferences.

---

## 🎨 Design System
*   **Typography:** Poppins (Friendly, modern, and readable)
*   **Color Palette:**
    *   Primary Accent: `#FF6B35` (Appetizing Vibrant Orange)
    *   Dark Neutral: `#1A1A1A` (Clean typography contrast)
*   **Key Features:** Smart search filters, dynamic order tracking timeline, and one-tap reorder widget.
